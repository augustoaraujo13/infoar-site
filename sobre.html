Projeto

Infoar

Infoar é uma empresa fictícia, criada para dar uma narrativa para esse projeto. A infoar é uma loja de e-commerce especializada na venda de periféricos de informática, fones de ouvidos, cabos, mouses, teclados, webcam e muitos mais através do nosso e-commerce, que contem vários produtos com preços acessíveis e de alta qualidade para os nossos clientes. Esse projeto é para fins de portfolio, para futuramente ser mostrado para recrutadores.

  • Infoar é uma empresa fictícia.
  • Esse projeto é para portfolio.
  • Mostrar aos recrutadores o meus conhecimentos.
